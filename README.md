# Calculadora Web

Vista básica de una calculadora desarrollada con HTML y CSS.

## Funcionalidades

- Ingreso de dos valores
- Selección de operación
- Visualización de resultado
- Selectores preparados para testing UI

## Cómo ejecutar

1. Abrir el archivo `index.html` en el navegador.

## Verificación de selectores

Se verificaron los selectores CSS utilizando DevTools y `document.querySelector()`.

Ejemplo:

```javascript
document.querySelector('#first-value')
```
